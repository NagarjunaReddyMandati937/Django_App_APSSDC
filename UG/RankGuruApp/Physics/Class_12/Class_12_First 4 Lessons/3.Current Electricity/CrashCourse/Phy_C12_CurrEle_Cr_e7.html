    \[ { R }^{ ' }\quad and\quad (R<{ R }^{ ' }) \]

    , then the resultant resistance lies between R/2 and R.
    If

    \[ { R }^{ ' } \]

    is only slightly greater than R, the resistance is a little greater than R/2 and if

    \[ { R }^{ ' } \]

    is very large as compared to R, resultant resistance almost approaches the value of R.
    In this case, let R = 0.01 ohm and

    \[ { R }^{ ' } \]

    = 1000 ohms.
    Here,

    \[ { R }^{ ' } \]

    is very large as compared to R so, resultant resistance almost approaches the value of R, that is, 0.01 ohm.
