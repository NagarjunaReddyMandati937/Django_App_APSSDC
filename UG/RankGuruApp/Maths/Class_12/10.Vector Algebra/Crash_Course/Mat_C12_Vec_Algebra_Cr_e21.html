Let

    \[ \vec{r}=a\hat{i}+b\hat{j}+c\hat{k} \]


    Then,


    \[ \vec{r}\cdot \hat{i}=a \]


    \[ \vec{r}\cdot \hat{j}=b \]


    \[ \vec{r}\cdot \hat{k}=c \]


    Now,


    \[ \left ( \vec{r}\cdot \hat{i} \right )\left ( \hat{i} \times \vec{r}\right )+\left ( \vec{r}\cdot \hat{j} \right )\left ( \hat{j} \times \vec{r}\right )\]


    \[+\left ( \vec{r}\cdot \hat{k} \right )\left ( \hat{k} \times \vec{r}\right ) \]


    \[ =a\left ( \hat{i} \times \vec{r}\right )+b\left ( \hat{j} \times \vec{r}\right )+c\left ( \hat{k} \times \vec{r}\right ) \]


    \[ =a(b\hat{k}-c\hat{j})+b(-a\hat{k}+c\hat{i})\]


    \[+c(a\hat{j}-b\hat{i}) \]


    \[ =0 \]
