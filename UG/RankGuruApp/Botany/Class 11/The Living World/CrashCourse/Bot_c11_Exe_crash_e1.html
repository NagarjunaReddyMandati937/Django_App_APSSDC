• According to the binomial system of classification, each name has two components, the generic name and the specific epithet.

  • Linnaeus introduced the binomial system of classification.

  • The first part of the name identifies the genus to which the species belongs, the second part, the specific name or specific epithet which identifies the species within the genus.

  • According to the binomial system of classification in Mangifera indica, the word Mangifera is a genus.
